# TEXT RECOGNIZER
==================

This is an app elaborating camera feature to take image and use the result to be scanned using MLKit Text Recognition.

<p>What're inside?
<br>- Jetpack Compose
<br>- Hilt
<br>- Firestore
<br>- MLKit - Text Recognition
</p>
